# Athlete Mindfulness & Performance Analytics Dashboard

An interactive, data-driven analytics platform built to explore the quantitative impacts of mindfulness, psychological stress mitigation, and sleep quality on athletic output metrics.

## Key Technical Features
- **Data Modeling:** Simulates realistic, multi-variable athlete biometrics with logical trend correlations.
- **Interactive Dashboards:** Built using Streamlit to offer real-time data slicing by Athlete ID and specific dates.
- **Advanced Visualizations:** Uses Plotly to generate dynamic correlation scatter plots and time-series trends.

## Tech Stack
- **Language:** Python
- **Libraries:** Streamlit, Pandas, NumPy, Plotly
- **Deployment:** Streamlit Community Cloud
